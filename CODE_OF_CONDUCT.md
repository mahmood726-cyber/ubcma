# Code of Conduct

This project follows the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

The full text and rationale live at that URL.

## Enforcement

To report a Code of Conduct concern, contact the maintainer:

- Email: **mahmood.ahmad2@nhs.net** with subject `[CONDUCT]`

The maintainer will acknowledge within 7 days and respond within 30 days.
Reports are kept confidential.

## Scope

This applies to all project spaces (issues, pull requests, discussions,
maintainer email). Anyone interacting in those spaces — contributors,
maintainers, casual users — is bound by the linked Covenant text.
